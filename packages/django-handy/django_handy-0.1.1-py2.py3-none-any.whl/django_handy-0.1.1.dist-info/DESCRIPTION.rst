Set of handy helpers for developing Django web applications.

Documentation
-------------

The full documentation is at https://django-handy.readthedocs.io.

Quickstart
----------

Install Django Handy::

    pip install django-handy

Add it to your `INSTALLED_APPS`:

.. code-block:: python

    INSTALLED_APPS = (
        ...
        'django_handy',
        ...
    )
