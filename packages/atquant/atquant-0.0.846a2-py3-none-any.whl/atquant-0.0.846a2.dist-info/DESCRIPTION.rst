=======
atquant
=======

atquant 从数据获取、算法交易、回测引擎，实盘模拟，实盘交易到数据分析，为程序化交易者提供了全套解决方案。
atquant 需要配合Auto-Trader(简称AT)使用，当AT运行时，可以使用atquant模块进行回测，回放，实盘操作。
